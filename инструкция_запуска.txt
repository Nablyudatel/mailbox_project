Для запуска программы необходимо:

1. Поставить python 3.8
2. Установить зависимости -> pip install requirements.txt
3. Через консоль зайти в текущую папку.
4. В текущем каталоге набрать -> python manage.py runserver
5. Перейти по адресу -> 127.0.0.1:8000/admin
6. Войти с помощью данных -> логин: admin@mail.ru, пароль: adminadmin
7. Перейти по адресу -> 127.0.0.1:8000
8. И продолжать работать с системой.
