Для запуска программы необходимо:

1. Поставить третий питон.
2. Установить библиотеку джанго -> pip install django
3. Через консоль зайти в текущую папку.
4. В текуще дирректории набрать -> python manage.py runserver
5. Перейти по адресу -> 127.0.0.1:8000/admin
6. Войти с помощью данных -> логин: admin, пароль: adminadmin
7. Перейти по адресу -> 127.0.0.1:8000
8. И продолжать работать с системой.
